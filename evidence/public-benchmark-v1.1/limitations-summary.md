# Public Benchmark v1.1 Limitations Summary

The current `v1.1` release remains limited.

Main limitations:

- only `4` baselines are released in this increment;
- literary prose still dominates the release, even after adding one government baseline;
- the released waves expand language coverage, but not yet to the target scale described in the benchmark plan;
- this release should still be treated as an intermediate release step rather than as a broad benchmark expansion result.
