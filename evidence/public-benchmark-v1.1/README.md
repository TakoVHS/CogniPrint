# Public Benchmark v1.1 Evidence Scaffold

This directory records the first public benchmark evidence increment after the currently released subset in `evidence/public-benchmark-v1/`.

It exists so that benchmark-expansion work can be tracked without mutating the meaning of the current `v1` release.

## Intended contents

The next increment should eventually contain:

- `manifest.json`
- `counts.json`
- `methods-summary.md`
- `results-summary.md`
- `limitations-summary.md`
- `evidence-table.md`
- `provenance-summary.md`

## Current status

The current `v1.1` release now contains four approved waves:

- `9` baseline excerpts;
- `54` controlled variants;
- `5` released languages (`de`, `ru`, `en`, `fr`, `es`);
- `3` released source classes (`public-domain literary text`, `public-domain government text`, `public-domain political prose`).

The release is materially more balanced than the earlier waves, but it remains a benchmark-growth layer rather than a broad benchmark validation result.
