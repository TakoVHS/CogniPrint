# Empirical Evidence Summary v1

This document gives a public, colleague-readable summary of the current CogniPrint empirical evidence package.

## Status statement

CogniPrint currently has a working empirical evidence package supporting a follow-up manuscript. The public research state consists of a mathematical manuscript layer, a reproducible local workstation, controlled perturbation campaigns, provenance-linked inputs, dataset scaffolds, and manuscript-oriented empirical drafting outputs.

The present materials should be read as working empirical materials for scientific review and manuscript preparation. They should not be presented as a publication-ready empirical paper.

## What exists now

The current project materials include:

- a mathematical framework for compact statistical text profiles;
- a local research workstation for controlled perturbation campaigns;
- source provenance records and validation;
- dataset scaffolds linked to campaign inputs;
- multi-campaign summaries and CSV/JSON exports;
- colleague-facing share packs;
- clean Methods, Results, and Limitations drafting materials for a follow-up manuscript;
- a local follow-up manuscript draft assembled from the current evidence package;
- a released public benchmark subset with explicit provenance and perturbation-axis metadata;
- an expanded descriptive statistical validation layer built from real campaign artifacts and benchmark coverage.

## Methods summary

The current workflow uses controlled perturbation campaigns. Each campaign is organized around a baseline text, a light edit, and a set of variants derived from the same topic. Variants are designed to change one main perturbation axis where practical, such as punctuation, lexical substitution, sentence segmentation, word order, compression, expansion, style register, stronger paraphrase, or cross-lingual rendering.

For each campaign, the workstation records profile shifts, metric deltas, campaign summaries, structured JSON/CSV outputs, LaTeX-oriented tables, and manuscript-oriented empirical notes. Source provenance is recorded separately so that input status and storage scope remain explicit.

## Results summary

The current outputs provide campaign-level evidence about how compact statistical profiles respond to controlled text perturbations. The evidence package supports discussion of observed patterns, measured shifts, profile differences, perturbation effects, and stability tendencies across the present campaigns.

The current outputs are useful for colleague review because they provide explicit artifacts rather than only informal impressions. They also provide a basis for drafting a more structured empirical section in a follow-up manuscript.

No broad corpus-level claim is made from the current material. The present outputs document observed behavior inside the controlled campaigns that have been run so far.

The current validation layer adds bootstrap mean intervals, axis-level ablation summaries, repeatable multi-draw random baseline summaries, threshold sensitivity outputs across cosine, Euclidean, and Manhattan metrics, and richer benchmark-versus-campaign bridge summaries. These remain descriptive and should not be read as inferential certainty claims.

## Limitations summary

The current evidence package remains bounded by several limitations:

- the corpus is still small;
- the current texts are primarily self-authored local research material;
- the campaign design is controlled but not yet broad across heterogeneous corpora;
- the outputs describe profile behavior only;
- the current materials are intended for research review and manuscript preparation;
- further work requires larger corpora, more heterogeneous domains, and stronger comparative baselines.
- the current statistical layer is still descriptive and does not constitute a completed inferential program.
