# Post-arXiv Update Checklist

This checklist should be executed immediately after the first public arXiv identifier becomes available.

## Update the website

1. Replace `arXiv submission pending` with the live arXiv identifier on:
   - `https://cogniprint.org/`
   - `https://cogniprint.org/papers/`
2. Add the arXiv abstract page URL to the paper links.
3. Keep the manuscript title and author form unchanged.

## Update repository metadata

1. Update `README.md` if the citation example still says `arXiv submission pending`.
2. Update `CITATION.cff` if a public article URL should be added.
3. Verify Zenodo release metadata remain aligned with the same canonical title and author identity.

## Update public profiles

1. Add the manuscript to ORCID.
2. Add the manuscript to Google Scholar once indexed.
3. Add the manuscript to ResearchGate using the same canonical metadata.
