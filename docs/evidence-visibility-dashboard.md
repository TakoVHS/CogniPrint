# Evidence Visibility Dashboard

The evidence visibility dashboard is the static website counterpart of the TypeScript evidence-visibility generator.

## Public path

- `/evidence/dashboard.html`

## Intended use

The page provides a compact view of:

- the current empirical snapshot;
- the first released public benchmark subset;
- the descriptive validation v1.1 layer.

It is meant for:

- reviewer orientation;
- due-diligence scanning;
- manuscript support.

It is not meant to present a stronger claim than the underlying evidence package already supports.

## Required public framing

Use the phrase:

> working empirical evidence package supporting a follow-up manuscript

Do not describe the dashboard as evidence of:

- a completed empirical study;
- forensic determination;
- guaranteed source classification.
