# Repository Frontpage Note

This note is intended to clarify how the current CogniPrint repository should be read by colleagues and reviewers.

## Current status

CogniPrint currently contains a mathematical manuscript layer, a reproducible local empirical workstation, provenance-aware input records, controlled perturbation campaign workflows, dataset scaffolds, multi-campaign summaries, colleague-facing share packs, and manuscript-oriented drafting outputs.

The most concise current status is:

> CogniPrint has a working empirical evidence package supporting a follow-up manuscript.

## Where to start

For a quick review, start with:

1. `docs/empirical-evidence-summary-v1.md`
2. `docs/current-state-summary.md`
3. `docs/public-vs-local-materials.md`
4. `docs/colleague-communication-pack.md`

## Empirical base

The current empirical summary records:

- 5 controlled perturbation campaigns;
- 41 comparison rows;
- provenance-linked inputs;
- dataset scaffolds;
- campaign summaries;
- table-ready outputs;
- clean Methods, Results, and Limitations drafting materials.

## Framing rule

Describe the current state as a working empirical evidence package and a research programme in manuscript consolidation.

Do not describe it as a settled publication-level empirical paper.
