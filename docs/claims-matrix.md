# Claims Matrix

## Supported claims

- CogniPrint is a reproducible research workstation for compact statistical profiles of text.
- CogniPrint supports controlled perturbation campaigns with provenance-linked local artifacts.
- CogniPrint exposes a public empirical evidence snapshot, a released benchmark subset, and a descriptive validation layer.
- CogniPrint produces manuscript-oriented and reviewer-oriented outputs.

## In-progress validation claims

- The current package shows observed pattern differences across controlled perturbation families.
- The benchmark-linked validation layer supports review of random-reference summaries, threshold sensitivity, and benchmark-versus-campaign bridge material.
- Stronger inferential validation remains a next-step task rather than a completed result.

## Explicit non-claims

- CogniPrint is not a forensic or attribution decision system.
- CogniPrint does not provide confirmed identity judgments.
- CogniPrint does not provide guaranteed classification.
- CogniPrint does not provide final judgment outputs about a text.
- The current public package is not a publication-ready empirical paper.
- The optional hosted app layer does not imply stronger scientific conclusions than the research core supports.
