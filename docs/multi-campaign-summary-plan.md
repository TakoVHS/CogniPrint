# Multi-Campaign Summary Plan

This document records the recommended next step after the current CogniPrint empirical baseline.

## Objective

Accumulate several real campaign-level runs and produce a higher-level summary that can support:

- internal scientific review;
- manuscript planning;
- appendix preparation;
- future dataset publication work;
- colleague-facing MVP materials.

## Current state

The current empirical base should always be described honestly in terms of the number of real campaign directories and comparison rows currently available. Do not write as if a larger corpus already exists.

## Minimum target before a stronger second-paper draft

Aim to collect:

- 3 to 5 real campaign directories;
- a markdown summary for each campaign;
- a manuscript appendix file for each campaign;
- at least one LaTeX table per campaign;
- one cleaned dataset scaffold linked to a representative campaign;
- refreshed colleague-facing share packs from real artifacts.

## Recommended sequence

1. Maintain provenance in `workspace/input/SOURCES.md`.
2. Prepare a genuinely separate baseline and controlled variants for the next campaign.
3. Run the next real campaign.
4. Refresh `cogniprint campaign summarize-all`.
5. Refresh colleague-facing share packs.
6. Refresh paper-2 drafting outputs.
7. Convert repeated observations into second-paper draft sections.

## Safe interpretation rule

The multi-campaign summary should speak in terms of:

- repeated observed patterns;
- metric shifts;
- profile differences;
- perturbation effects;
- stability tendencies.

Avoid stronger language implying certainty or definitive attribution.
