# arXiv Submission Plan

## Purpose

This document records a conservative submission plan for the first public preprint associated with CogniPrint.

## Working title

**Cognitive Fingerprints: A Mathematical Framework for Statistical Profiling of Text**

## Recommended initial categories

Primary candidates:
- `math.ST`
- `cs.CL`

The final category choice should follow the dominant balance of the manuscript:
- use `math.ST` if the paper is primarily framed through statistical structure, formal assumptions, and mathematical analysis;
- use `cs.CL` if the paper is primarily framed through text representation, corpus protocol, and language-focused experimentation.

## Minimum package before submission

The first arXiv-ready package should include:
- a stable title;
- a consistent author identity;
- abstract, introduction, formal setting, stability section, and limitations section;
- a clearly separated empirical protocol section;
- references, even if initially minimal;
- a short appendix if notation or proof details become too long for the main flow.

## Submission rule

No sentence in the public preprint should make a stronger claim than one of the following supports:
- a theorem with explicit assumptions;
- a proof or proof sketch;
- a documented empirical result;
- a clearly labelled open question.

## Endorsement preparation

If endorsement is required, preparation should include:
- a 3-4 sentence description of the manuscript;
- a concise explanation of the mathematical contribution;
- a link to the repository and project summary;
- a short, polite endorsement request template.

## Recommended sequencing

1. Finalise the draft text.
2. Freeze author metadata across repository files.
3. Prepare the arXiv manuscript source.
4. Run a final terminology audit.
5. Submit only after the manuscript and metadata match exactly.
