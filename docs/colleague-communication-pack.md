# Colleague Communication Pack

This document provides concise language for presenting the current CogniPrint MVP to scientific colleagues.

## One-sentence description

CogniPrint is a reproducible research workstation for studying compact statistical profiles of text and measuring how those profiles shift under controlled perturbations.

## Short spoken description

CogniPrint is not being presented as a forensic or authorship-certainty system. At the current stage, it is a local empirical research workstation that runs controlled perturbation campaigns, records profile shifts and metric deltas, and produces campaign-level summaries, notes, tables, and dataset scaffolds for further scientific work.

## What to emphasize

- reproducible local workflow;
- controlled perturbation studies;
- profile shifts and stability tendencies;
- explicit artifacts: summaries, appendix text, LaTeX tables, dataset manifests;
- cautious interpretation limits.

## What not to claim

Do not describe CogniPrint as:

- proof of authorship;
- forensic certainty engine;
- definitive source classifier;
- final judgment system.

## Recommended package for colleagues

Share a compact set of materials:

1. `workspace/share/colleague-pack-003/README.md`
2. `workspace/share/colleague-pack-003/project-summary.md`
3. one real `campaign-summary.md`
4. one real `manuscript-appendix.md`
5. one LaTeX summary table
6. one dataset-manifest summary
7. one short note on interpretation limits

## Suggested discussion framing

Useful phrases:

- observed pattern
- measured shift
- profile difference
- perturbation effect
- stability tendency
- campaign-level evidence

Avoid stronger phrases implying certainty.
