## Canonical title

**Cognitive Fingerprints: A Mathematical Framework for Statistical Profiling of Text**

## Canonical short abstract posture

The manuscript presents a mathematical framework for compact statistical profiling of text and studies profile similarity and perturbation stability under a research framing. It should not be positioned as a definitive attribution or forensic instrument.

## Package checklist

Before submission, confirm:

- `paper/main.tex` is final;
- `paper/references.bib` is final;
- local PDF compiles cleanly;
- author spelling is fixed as `Adriashkin Roman`;
- public-facing metadata match `CITATION.cff` and the website;
- no prohibited legacy names remain;
- the manuscript still uses cautious, research-only wording.

## Submission field checklist

- **Title:** use the manuscript title exactly;
- **Authors:** Adriashkin Roman;

- **Affiliation:** CogniPrint Research Initiative;
- **Categories:** math.ST as primary, cs.CL as secondary;

- **Journal reference:** leave empty unless a real venue exists;

## Immediate post-submission actions

Once the submission is accepted and a public arXiv identifier exists:

1. update `cogniprint.org` to replace `arXiv submission pending`;
2. update `papers/` page with the live arXiv abstract URL;
3. update `README.md` and citation examples if needed;
4. add the work to ORCID;
5. add the work to Google Scholar once indexed;
6. add the work to ResearchGate using the same canonical metadata.
