# CogniPrint Content Scanner API

FastAPI service for the commercial CogniPrint Content Scanner SaaS layer.

This app wraps the existing deterministic CogniPrint research engine and exposes a production API for scanning text, enforcing usage quotas, and integrating subscription billing.

Outputs remain statistical and heuristic signals only. They are not legal conclusions, source guarantees, authorship guarantees, or final judgments about text.
