# Site Layer

This directory contains the minimal public website layer for `cogniprint.org`.

## Purpose

The website is intended to function as the public research-facing entry point for:
- the CogniPrint Research Initiative;
- the manuscript and preprint layer;
- the software repository;
- the Zenodo archive and ORCID identity.

## Recommended deployment

The preferred deployment target is GitHub Pages with a custom domain.

## Expected files

- `index.html`
- `research.html`
- `papers/index.html`
- `software/index.html`
- `disclaimer.md`
- `404.html`
- `robots.txt`
- `sitemap.xml`
- `CNAME`
- `.nojekyll`
